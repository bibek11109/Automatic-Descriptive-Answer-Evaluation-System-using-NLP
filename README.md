# Automatic-Descriptive-Answer-Evaluation-System-using-NLP
This project is used to grade answers automatically using AI, the task is done automatically reducing the time for manual process of evaluation. 
We will have students and teachers portal where students can add their answers for grading and evaluation whereas the teachers can create, update, delete the questions.
Moreover, the BERT model has been used for training with over 12000 descriptive answer datasets.
